# PRD: 작물 추천 위치 변경

## 📋 개요
작물 추천 기능의 진입점 위치를 변경하여 사용자 접근성을 개선합니다.

**변경 이유**: 사용자가 작물 추천 기능을 더 쉽게 찾고 접근할 수 있도록 진입점 위치를 최적화합니다.

---

## 🔄 변경 내용

### 기존 위치
- **홈 화면**: 날씨 위젯과 함께 좌우 배치된 카드 형태의 배너
  - 위치: `HomePage.tsx` (334-357번째 줄)
  - 레이아웃: 그리드 2열 구조 (날씨 위젯 | 작물 추천 배너)
  - 버튼: "작물 추천 받으러가기" 버튼 클릭 시 `/recommendations/input` 이동

### 변경 후 위치
- **옵션 1**: 모바일 네비게이션에 추가 (권장)
  - 하단 네비게이션 바에 "작물 추천" 탭 추가
  - 4개 탭에서 5개 탭으로 확장 (홈, 농장&작물, 작물추천, 캘린더, 마이)

- **옵션 2**: 헤더에 버튼 추가
  - 상단 헤더에 작물 추천 아이콘/버튼 추가
  - 모든 페이지에서 접근 가능

- **옵션 3**: 홈 화면 상단으로 이동
  - 날씨 위젯과 분리하여 독립 섹션으로 상단 배치
  - 더 눈에 띄는 위치로 이동

## 📐 기술 스펙

### 옵션 1: 모바일 네비게이션 추가 (권장)

#### 파일 수정
**파일**: `FarmMate/src/components/layout/mobile-nav.tsx`

**변경 사항**:
```typescript
// 기존: grid-cols-4
// 변경: grid-cols-5

// 새로운 탭 추가
import { Sprout } from 'lucide-react';

<Link href="/recommendations/input" className={`flex flex-col items-center py-3 px-1 ${
  isActive("/recommendations") ? "text-primary bg-primary/10" : "text-gray-600"
}`}>
  <Sprout className="w-5 h-5" />
  <span className="text-xs mt-1">작물추천</span>
</Link>
```

#### 네비게이션 레이아웃
- 기존: 4개 탭 (홈, 농장&작물, 캘린더, 마이)
- 변경: 5개 탭 (홈, 농장&작물, 작물추천, 캘린더, 마이)
- 탭 순서: 홈 → 농장&작물 → 작물추천 → 캘린더 → 마이

#### 홈 화면 배너 처리
- **옵션 A**: 홈 화면 배너 제거 (네비게이션으로 완전 이동)
- **옵션 B**: 홈 화면 배너 유지 (중복 진입점 제공)

## 🎨 UI/UX 고려사항

### 모바일 네비게이션 (옵션 1)
- **장점**: 
  - 모든 페이지에서 쉽게 접근 가능
  - 기능의 중요도가 높아짐
  - 사용자 학습 비용 낮음
  
- **단점**:
  - 네비게이션 바가 5개로 늘어나 밀집도 증가
  - 작은 화면에서 탭 크기 축소 가능성

- **해결책**:
  - 탭 아이콘 크기 및 간격 최적화
  - 텍스트 레이블 간결화 ("작물추천" → "추천" 또는 아이콘만)

### 헤더 버튼 (옵션 2)
- **장점**:
  - 데스크톱에서 접근성 좋음
  - 기존 레이아웃 변경 최소화

- **단점**:
  - 모바일에서 헤더 공간 제약
  - 홈 화면 배너와 중복 가능

### 홈 화면 상단 (옵션 3)
- **장점**:
  - 홈 화면에서 명확한 진입점
  - 큰 배너로 시각적 강조

- **단점**:
  - 다른 페이지에서는 접근 어려움
  - 홈 화면 진입 의존적
