# 📄 PRD Template (개발용)

| 항목 | 내용 |
|------|------|
| **기능 이름** | 작물 추천 기능 |

| **개요** | 홈 화면에서 작물 추천 화면으로 이동하고, 원하는 기간에 대한 최적 작물을 추천 받을 수 있도록 한다. |

| **요구사항** |  |
Phase 1: 진입점 & 네비게이션

- [x] 홈 화면에 "작물 추천 받으러 가기" 버튼 추가

- [x] 버튼 클릭 시 작물 추천 화면으로 이동

Phase 2: 입력 화면 (작물 추천 조건 설정)

- [x] 재배 위치 선택 (노지, 시설, 기타)

- [x] 재배 범위 입력 (소면적 단위 입력 가능, 예: 100㎡ 이하 등)

- [x] 재배 시작 시기 (월 단위) 선택

- [x] 재배 종료 시기 (월 단위) 선택

- [x] 조건 입력 완료 후 "작물 추천" 버튼 클릭 → 알고리즘 실행

Phase 3: 처리 & 로딩 화면

- [x] "작물 추천 중입니다. 잠시만 기다려 주세요" 로딩 애니메이션 표시

- [x] 파이썬 알고리즘 실행 및 결과 Fetch

Phase 4: 결과 리스트 화면

- [x] 추천된 작물들을 리스트 카드 형태로 표시

작물명 + 대표 이미지

예상 생산량 / 예상 수익 / 예상 비용

재배 난이도 & 참고 정보 표시

- [x] summary / detail 탭 전환 가능

- [x] **뱃지 시스템 구현**
  - 첫 번째 추천 결과 (Plan A): ⭐ 종합 추천 뱃지 (보라-남색 그라디언트)
  - 예상 수익이 가장 높은 조합: 💰 최고 수익 뱃지 (주황-호박색 그라디언트)
  - 두 뱃지는 서로 다른 조합에 나타날 수 있음

- [x] **추천 조합 더보기 기능**
  - 기본 3개 조합 표시
  - 더보기 클릭 시 최대 6개 조합까지 표시
  - 모든 조합 확인 시 "모든 추천 조합을 확인했습니다 ✓" 안내 문구 표시

Phase 5: 상세 결과 화면

- [x] 특정 추천 작물 선택 시 상세 화면 표시

작물명, 품목 구성, 예상 생산/수익/비용

세부 스펙(노동일정, 재배 난이도 등)

- [ ] "플래너 보기" 버튼 제공

- [ ] 클릭 시 해당 작물이 플래너 모듈과 연동되어 저장

Phase 6: 예외 처리 & UX 개선

- [x] 조건 입력이 불완전할 경우 → 경고 메시지 표시

- [x] 추천 결과가 없을 경우 → "조건에 맞는 작물이 없습니다" 안내

- [x] **라디오 버튼 선택 시스템**
  - 각 추천 카드에 라디오 버튼 추가
  - 사용자가 원하는 조합 선택 가능
  - 선택된 카드에 링 효과 및 그림자 효과 적용

- [x] **추천 결과 저장 기능**
  - 선택한 조합을 로컬 스토리지에 저장
  - "추천 결과 저장하기" 버튼 제공
  - 농장 정보와 함께 결과 저장

- [x] **추천 기록 보기 기능**
  - 이전 추천 결과를 확인할 수 있는 히스토리 페이지
  - 각 기록별 상세 정보 확인 가능


| **우선순위** | High (핵심 기능) |

| **비고** | 
- 기술 스택: React, TypeScript, Supabase Edge Functions
- 추천 알고리즘: MILP (Mixed Integer Linear Programming) 알고리즘 사용
- 점수 산정 기준: 수익성, 노동편의성, 품종희소성의 가중평균
- 뱃지 표시 기준:
  - 종합 추천: 알고리즘 점수 가장 높은 조합 (첫 번째 추천 결과)
  - 최고 수익: 예상 매출액이 가장 높은 조합
- 더보기 기능: 기본 3개 → 최대 6개 조합 표시
