# 📄 PRD Template (개발용)

| 항목 | 내용 |
|------|------|
| **기능 이름** | 작물 추천 기능 |

| **개요** | 홈 화면에서 작물 추천 화면으로 이동하고, 원하는 기간에 대한 최적 작물을 추천 받을 수 있도록 한다. |

| **요구사항** |  |
Phase 1: 진입점 & 네비게이션

- [ ] 홈 화면에 "작물 추천 받으러 가기" 버튼 추가

- [ ] 버튼 클릭 시 작물 추천 화면으로 이동

Phase 2: 입력 화면 (작물 추천 조건 설정)

- [ ] 재배 위치 선택 (노지, 시설, 기타)

- [ ] 재배 범위 입력 (소면적 단위 입력 가능, 예: 100㎡ 이하 등)

- [ ] 재배 시작 시기 (월 단위) 선택

- [ ] 재배 종료 시기 (월 단위) 선택

- [ ] 조건 입력 완료 후 "작물 추천" 버튼 클릭 → 알고리즘 실행

Phase 3: 처리 & 로딩 화면

- [ ] "작물 추천 중입니다. 잠시만 기다려 주세요" 로딩 애니메이션 표시

- [ ] 파이썬 알고리즘 실행 및 결과 Fetch

Phase 4: 결과 리스트 화면

- [ ] 추천된 작물들을 리스트 카드 형태로 표시

작물명 + 대표 이미지

예상 생산량 / 예상 수익 / 예상 비용

재배 난이도 & 참고 정보 표시

- [ ] summary / detail 탭 전환 가능

Phase 5: 상세 결과 화면

- [ ] 특정 추천 작물 선택 시 상세 화면 표시

작물명, 품목 구성, 예상 생산/수익/비용

세부 스펙(노동일정, 재배 난이도 등)

- [ ] "플래너 보기" 버튼 제공

- [ ] 클릭 시 해당 작물이 플래너 모듈과 연동되어 저장

Phase 6: 예외 처리 & UX 개선

- [ ] 조건 입력이 불완전할 경우 → 경고 메시지 표시

- [ ] 추천 결과가 없을 경우 → "조건에 맞는 작물이 없습니다" 안내

- [ ] 테스트 모드(더미 데이터)로 결과 미리보기 가능
